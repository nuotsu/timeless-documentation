The _Footer Bottom_ section is a global section displayed on all pages that is used to display the country/currency and language selectors, payment type icons, and copyright information. Customizations include a color scheme and showing [Follow on Shop](https://help.shopify.com/en/manual/online-store/themes/customizing-themes/follow-on-shop).
